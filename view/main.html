Taipei Public Safety Visualization Website

台北市犯罪資料視覺化網站

   In this project, we want to visualize the relationship between the number of population, police stations, street lights,street monitors, and the number of burglaries that happen in each district in Taipei city.
We expect to create a heatmap with several graphs (such as barplots, line charts) besides the map to further explore the relationships.
The heatmap is based on the ratio between the number of crimes and the number of population, police stations, street lights, street monitors…, etc.
We will possibly add more datasets as we will keep searching for more useful ones.
